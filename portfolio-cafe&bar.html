CAFE&BARを想定した架空サイトのトップページのデザインとコーディング

目的
お洒落で大人な雰囲気の隠れ家的お店を知ってもらうため。
ターゲット
落ち着いた空間でゆったりとした時間を過ごしたい20代〜40代の方
コーヒーやお酒を楽しみながら、特別なひとときを求める大人の方。
工夫した点
落ち着いた雰囲気と温かみを感じられるデザインを意識し、写真を活かして居心地の良さが伝わるよう仕上げています。
また、PC版ではメニュー部分にブラインドエフェクトを取り入れ、お店の雰囲気を視覚的に表現しています。
制作期間
約４週間(企画・デザイン制作２週間、コーディング２週間)
使用言語・ツール
Figma / HTML / CSS / JavaScript
